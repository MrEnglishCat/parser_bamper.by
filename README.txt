1 Предварительно нужно установить интепретатор python(проект писался на версии 3.12)
    https://www.python.org/downloads/
    скачиваем последнюю версию. На текущий момент 3.12.3

2 Устанавливаем IDE(писал в PyCharm):
    PyCharm Community Edition:
        https://www.jetbrains.com/pycharm/download/?section=windows

    тут ссылка под Windows, но можно и под другую ОС
    PS можно и в другой IDE поддерживающей python.
    PPS в PyCharm мне будет проще объяснить т к в основном в нем работаю

3 далее возможныварианты либо используя IDE делать либо терминал.
  Опишу вариант запуска через IDE(через терминал на винде сталкивался с ограничениями
  оболочки на запуск скриптов)

4 Открыть IDE создать новый проект, выбрав использовать локальный интерпретатор
     в результате IDE создаст проект и виртуальное окружение для него(для того что бы не
     засорять глобальный интерпретатор)

5 Скопировать/переместить файлы bamper_by.py & requirements.txt в корневую директорию
 проекта
6 открыть терминал в IDE(на последней строке должна быть запись (.venv) - значит виртуальное окружение
    активировано). Терминал открывает обычно сразу директорию проекта.
    прописать команду:
       pip install requirements.txt

       IDE должна установить нужные библиотеки из файла requirements.txt
7 далее в принципе можно запускать файл bamper_by.py на исполнение

8 в терминал выводятся информационные сообщения(если будет необходимость можно записать в файл/ так же можно заменить
    текст самих сообщений), что бы можно было понимать, что скрипт работает =)
